A [Flarum](https://flarum.org) extension. flarum ext blacklist

## Installation

Install with composer:

```sh
composer require mattoid/flarum-ext-blacklist :"*"
```

## Updating

```sh
composer update mattoid/flarum-ext-blacklist :"*"
php flarum migrate
php flarum cache:clear
```
